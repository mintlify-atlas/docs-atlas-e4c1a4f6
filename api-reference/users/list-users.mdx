---
title: 'List Users'
api: 'GET /v1beta1/users'
---

Retrieve a list of users with optional filtering by organization, group, state, or keyword search.

## Authentication

This endpoint requires authentication. Include a valid bearer token in the Authorization header.

## Query Parameters

<ParamField query="page_size" type="integer">
  Number of users to return per page. Used for pagination.
</ParamField>

<ParamField query="page_num" type="integer">
  Page number to retrieve. Used for pagination.
</ParamField>

<ParamField query="keyword" type="string">
  Search keyword to filter users by name or email.
</ParamField>

<ParamField query="org_id" type="string">
  Filter users by organization ID. Returns only users belonging to the specified organization.
</ParamField>

<ParamField query="group_id" type="string">
  Filter users by group ID. Returns only users belonging to the specified group.
</ParamField>

<ParamField query="state" type="string">
  Filter users by state. Possible values:
  - `enabled` - Active users
  - `disabled` - Disabled users
</ParamField>

## Response

<ResponseField name="count" type="integer">
  Total count of users returned in the response.
</ResponseField>

<ResponseField name="users" type="array">
  Array of user objects.
  <ResponseField name="id" type="string">
    Unique identifier for the user (UUID).
  </ResponseField>
  <ResponseField name="name" type="string">
    User's username or slug.
  </ResponseField>
  <ResponseField name="title" type="string">
    User's title or display name.
  </ResponseField>
  <ResponseField name="email" type="string">
    User's email address.
  </ResponseField>
  <ResponseField name="avatar" type="string">
    URL to the user's avatar image.
  </ResponseField>
  <ResponseField name="state" type="string">
    Current state of the user account (`enabled` or `disabled`).
  </ResponseField>
  <ResponseField name="metadata" type="object">
    Custom metadata associated with the user.
  </ResponseField>
  <ResponseField name="created_at" type="timestamp">
    Timestamp when the user was created.
  </ResponseField>
  <ResponseField name="updated_at" type="timestamp">
    Timestamp when the user was last updated.
  </ResponseField>
</ResponseField>

## Example Request

<CodeGroup>

```bash cURL
curl -X GET 'https://api.frontier.example.com/v1beta1/users?page_size=10&org_id=org_123' \
  -H 'Authorization: Bearer <token>'
```

```javascript JavaScript
const response = await fetch('https://api.frontier.example.com/v1beta1/users?page_size=10&org_id=org_123', {
  method: 'GET',
  headers: {
    'Authorization': 'Bearer <token>'
  }
});
const data = await response.json();
```

```python Python
import requests

response = requests.get(
    'https://api.frontier.example.com/v1beta1/users',
    params={'page_size': 10, 'org_id': 'org_123'},
    headers={'Authorization': 'Bearer <token>'}
)
data = response.json()
```

</CodeGroup>
