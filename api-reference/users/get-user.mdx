---
title: 'Get User'
api: 'GET /v1beta1/users/{id}'
---

Retrieve detailed information about a specific user by their ID, email, or username.

## Authentication

This endpoint requires authentication. Include a valid bearer token in the Authorization header.

## Path Parameters

<ParamField path="id" type="string" required>
  User identifier. Can be one of:
  - User ID (UUID)
  - Email address
  - Username/slug
</ParamField>

## Response

<ResponseField name="user" type="object">
  The requested user object.
  <ResponseField name="id" type="string">
    Unique identifier for the user (UUID).
  </ResponseField>
  <ResponseField name="name" type="string">
    User's username or slug.
  </ResponseField>
  <ResponseField name="title" type="string">
    User's title or display name.
  </ResponseField>
  <ResponseField name="email" type="string">
    User's email address.
  </ResponseField>
  <ResponseField name="avatar" type="string">
    URL to the user's avatar image.
  </ResponseField>
  <ResponseField name="state" type="string">
    Current state of the user account (`enabled` or `disabled`).
  </ResponseField>
  <ResponseField name="metadata" type="object">
    Custom metadata associated with the user.
  </ResponseField>
  <ResponseField name="created_at" type="timestamp">
    Timestamp when the user was created.
  </ResponseField>
  <ResponseField name="updated_at" type="timestamp">
    Timestamp when the user was last updated.
  </ResponseField>
</ResponseField>

## Example Request

<CodeGroup>

```bash cURL (by ID)
curl -X GET 'https://api.frontier.example.com/v1beta1/users/9f256f86-4a1e-4b2a-9c45-6d2c8f5a3b7e' \
  -H 'Authorization: Bearer <token>'
```

```bash cURL (by email)
curl -X GET 'https://api.frontier.example.com/v1beta1/users/john.doe@example.com' \
  -H 'Authorization: Bearer <token>'
```

```javascript JavaScript
const userId = '9f256f86-4a1e-4b2a-9c45-6d2c8f5a3b7e';
const response = await fetch(`https://api.frontier.example.com/v1beta1/users/${userId}`, {
  method: 'GET',
  headers: {
    'Authorization': 'Bearer <token>'
  }
});
const data = await response.json();
```

```python Python
import requests

user_id = '9f256f86-4a1e-4b2a-9c45-6d2c8f5a3b7e'
response = requests.get(
    f'https://api.frontier.example.com/v1beta1/users/{user_id}',
    headers={'Authorization': 'Bearer <token>'}
)
data = response.json()
```

## Error Codes

<ResponseField name="400" type="error">
  Bad Request - Invalid user ID format.
</ResponseField>

<ResponseField name="401" type="error">
  Unauthorized - Invalid or missing authentication token.
</ResponseField>

<ResponseField name="404" type="error">
  Not Found - User does not exist with the provided identifier.
</ResponseField>

<ResponseField name="500" type="error">
  Internal Server Error - An unexpected error occurred.
</ResponseField>
