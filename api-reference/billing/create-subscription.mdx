---
title: 'Create Subscription'
openapi: 'POST /v1beta1/organizations/{org_id}/billing/{billing_id}/subscriptions'
---

<Note>
This endpoint creates a new subscription for a billing account. The subscription is tied to a specific plan.
</Note>

<RequestExample>

```bash cURL
curl -X POST "https://frontier.example.com/v1beta1/organizations/{org_id}/billing/{billing_id}/subscriptions" \
  -H "Authorization: Bearer <token>" \
  -H "Content-Type: application/json" \
  -d '{
    "plan_id": "plan_123",
    "skip_trial": false,
    "metadata": {}
  }'
```

```javascript JavaScript
const response = await fetch(
  'https://frontier.example.com/v1beta1/organizations/{org_id}/billing/{billing_id}/subscriptions',
  {
    method: 'POST',
    headers: {
      'Authorization': 'Bearer <token>',
      'Content-Type': 'application/json'
    },
    body: JSON.stringify({
      plan_id: 'plan_123',
      skip_trial: false,
      metadata: {}
    })
  }
);
const data = await response.json();
```

```python Python
import requests

response = requests.post(
    'https://frontier.example.com/v1beta1/organizations/{org_id}/billing/{billing_id}/subscriptions',
    headers={
        'Authorization': 'Bearer <token>',
        'Content-Type': 'application/json'
    },
    json={
        'plan_id': 'plan_123',
        'skip_trial': False,
        'metadata': {}
    }
)
data = response.json()
```
